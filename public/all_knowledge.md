

## 一、测量与运动

## 二、声现象

| 知识点 | 核心内容 |
|--------|----------|
| **声音的产生** | 一切发声体都在振动；振动停止，发声停止 |
| **声音的传播** | 需要介质（固、液、气）；真空不能传声；声速与介质种类和温度有关；回声 |
| **音调** | 频率决定音调；人耳听觉范围（20Hz~20000Hz）；超声波与次声波 |
| **响度** | 振幅决定响度；距离声源远近影响响度；分贝（dB） |
| **音色** | 发声体的材料、结构决定音色；用于辨别发声体 |
| **噪声** | 定义（物理角度与环保角度）；噪声等级；控制噪声的三途径（声源处、传播过程中、人耳处） |
| **声音的利用** | 传递信息（B超、声呐）；传递能量（超声波清洗、超声波碎石） |

## 三、光现象

| 知识点 | 核心内容 |
|--------|----------|
| **光源** | 天然光源与人造光源 |
| **光的直线传播** | 条件（同种均匀介质）；现象（影子、日食、月食、小孔成像）；光速（c=3×10⁸m/s） |
| **光的反射** | 反射定律（三线共面、两线分居、两角相等）；光路可逆；镜面反射与漫反射 |
| **平面镜成像** | 成像特点（等大、等距、垂直、虚像）；成像原理；应用 |
| **光的折射** | 折射规律（空气中的角大）；折射现象（筷子弯折、池底变浅、海市蜃楼） |
| **光的色散** | 白光由七色光组成；色光三原色（红、绿、蓝）；颜料三原色（品红、黄、青）；物体的颜色（透明与不透明） |
| **透镜** | 凸透镜与凹透镜；主光轴、光心、焦点、焦距；三条特殊光线 |
| **凸透镜成像** | u>2f（倒立缩小实像，照相机）；u=2f（倒立等大实像）；f<u<2f（倒立放大实像，投影仪）；u=f（不成像）；u<f（正立放大虚像，放大镜）；实像与虚像的区别 |
| **眼睛与视力矫正** | 眼睛的成像原理（晶状体=凸透镜，视网膜=光屏）；近视眼（成像在视网膜前，凹透镜矫正）；远视眼（成像在视网膜后，凸透镜矫正） |
| **显微镜与望远镜** | 基本结构原理 |

## 六、力与运动

| 知识点 | 核心内容 |
|--------|----------|
| **牛顿第一定律** | 内容；理想实验法（伽利略斜面实验）；力不是维持运动的原因，而是改变运动状态的原因 |

| **力与运动关系** | 不受力/受平衡力→静止或匀速直线运动；受非平衡力→运动状态改变（速度大小或方向变化） |
